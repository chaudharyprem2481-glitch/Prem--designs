How do I get a token? (2 minutes, one time)
  1. On GitHub, go to Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Click Generate new token. Give it any name, e.g. "Portfolio Admin".
  3. Set an expiration (a year is fine — you'll just repeat this step when it expires).
  4. Under Repository access, choose Only select repositories and pick your portfolio repo.
  5. Under Permissions → Repository permissions, find Contents and set it to Read and write.
  6. Click Generate token and copy it — GitHub only shows it once. Paste it above.
